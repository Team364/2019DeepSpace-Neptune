Default commands are specific commands that are to require a subsytsem when no
other commands are being run.

General open loop commands for individual talons can be found in prefabricated commands.

Periodic is intended to keep track of States that do not neccessarily run on commands.

Vision Periodic is intended to keep track of relevant vision information. Currently it is empty.

Drive Open Loop uses private drive states to decide what demand to send to the driveTrain open loop.
It is currently configured for Trigger Drive
