Subroutines run directly on button presses or axis input. They may start other commands.
These are all command groups meaning that they start other commands and may pass specific parameters to
the commands constructor.

Align is currently empty.

Climb is currently empty.

Elevate sets the arm and lift to a particular position using an integer representing a mode such as 
scoring on the three levels of the rocket or the cargo height for the cargo ship.

Retract puts the arm and grip in starting configuation so that damage to the appartus may be minimized.
It is to be accessed by both the driver and operator

Run grip either intakes or outtakes either cargo or a hatch depending on the trigger pulled and the current
object state.
